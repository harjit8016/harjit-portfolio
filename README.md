# Harjit Singh — Portfolio

Personal portfolio site for **Harjit Singh**, Senior iOS Developer.

Built with plain HTML, CSS, and vanilla JavaScript — NeoBrutalism design, zero build step, deploys directly to GitHub Pages.

## Local preview

Open `index.html` in a browser, or run a static server:

```bash
python3 -m http.server 8080
```

## Deploy (GitHub Pages)

Push to the `main` branch of your repo and enable Pages from the repo Settings → Pages → Source: `main` / root.
